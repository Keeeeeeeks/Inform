Zoom Manual

Zoom is an interpreter for Z-Code files, versions 3, 4, 5, 6, 7 and 8. These usually contain interactive fiction (IF), and typically have a filename ending in '.zX', where X is a number (for example, curses.z5). Zoom runs under the X-window system on POSIX systems, and also under Mac OS X. This file documents both versions (they are very similar). Z-Code games may also be contained in Blorb files (extension .blb or .zlb).
